1. 信号与图像基础

1.1 从空间域到频域

图像本质上是一个二维的离散信号，每个像素对应一个采样点的数值。灰度图像的每个像素，表示一个灰度值（0~255），彩色图像的每个像素，有多个通道（R/G/B）。像素值的分布就是空间域信息：

频域 描述的是 图像的变化频率，低频 表示平滑区域，也就是像素值之间的差异很小的地方，或者说变化缓慢；高频 表示边缘细节，相邻的像素之间数值差异很大，像素变化很快。

DFT 的中心点 F[0,0] 对应图像的直流分量，简单来说，就是图像的平均亮度，任何 自然图像 都会有非零平均亮度，因此 F[0,0] 值非常大，这就是为什么我们会看到频域的幅度分布图中，几乎全部亮度集中在中心。

高频、低频细节的幅度远小于直流分量，所以在幅度谱图上周围亮点显得很暗。而经过不同处理的图像，幅度谱的周围亮度，也发生了明显的变化。

平滑过后的图像，“十字形” 表现的更为明显，高频能量被抑制衰减，低频的能量更加的收敛。原图有轻微的 “十字形” 出现，而锐化后的图，外围高频被增强，幅度图发散的非常厉害。

1.2 DFT 的概念与应用

离散傅里叶变换（Discrete Fourier Transform, DFT） 是一种将离散信号转为频域表示的数学技术，它通过将信号分解为一系列正弦和余弦函数的和，来表示信号的频谱特征。

离散傅里叶变换（DFT）在信号处理中应用广泛，但很多术语源于一维信号（如音频）。针对二维图像数据，我们主要关注以下核心应用场景：

  1. 频域表示与频率分布分析：将图像从空间域转换到频域，通过观察频率分布（即幅度谱），分析图像的能量主要集中在低频（平滑区域）还是高频（边缘与噪声）。

  2. 特定频率成分的提取与抑制（滤波）：这是图像处理最核心的应用。通过设计滤波器，我们可以针对性地处理特定频率成分：

    • 低通：保留低频，抑制高频（用于去噪、平滑）。
    • 高通：保留高频，抑制低频（用于锐化、边缘检测）。
    • 带阻：专门去除周期性的干扰条纹（如去除摩尔纹）。

  3. 图像重构（IDFT）：在频域完成对频率成分的修改后，利用逆变换将数据还原回空间域，从而得到处理后的图像（如去噪后的清晰图、锐化后的细节图）。

  4. 压缩与编码基础：利用图像能量主要集中在低频的频域特征，舍弃人眼不敏感的高频细节，从而实现数据压缩（如 JPEG 标准的基础原理）。

1.3 一维 DFT 公式与解析

DFT 可以将一个长度为 N 的离散信号 x[n] 从时域变换到频域，表示为复数频谱 X[k]。数学上的定义为：

X[k]=∑n=0N−1x[n]⋅e−j2πNkn,k=0,1,...,N−1X[k]=\sum^{N-1}_{n=0}x[n]\cdot{e^{-j\frac{2\pi}{N}kn}}, \quad k=0,1,...,N-1 X[k]=n=0∑N−1​x[n]⋅e−jN2π​kn,k=0,1,...,N−1

  • x[n]x[n]x[n]：表示时域的离散信号
  • X[k]X[k]X[k]：表示频域的复数信号，包含幅度和相位信息
  • NNN：表示信号长度
  • e−j2πNkne^{-j\frac{2\pi}{N}kn}e−jN2π​kn：表示复指数基函数，等价于 cos⁡(2πknN)−jsin⁡(2πknN)\cos{(\frac{2\pi{kn}}{N})}-j\sin{(\frac{2\pi{kn}}{N})}cos(N2πkn​)−jsin(N2πkn​)，本质上是正弦与余弦的组合

来做一个小实验，我们生成一个正弦波（sin）作为主信号，再生成一段强度大约是正弦波 1/4 的噪声，两者进行结合，对复合信号进行离散傅里叶变换，看一看它的结果。

图 1.3 中第二行的 幅度谱（Magnitude） 表示 信号在各个频率上的强度。对于一个正弦波来说，由于波形固定，频率分量会不断叠加，因此在对应频率上出现一个明显的峰值。另一个峰值则是它的镜像（共轭对称性）。

相位谱（Phase） 表示 该频率分量在时域中的起始偏移位置，通常很难直观地从图中看出具体信息（因为噪声的相位是随机的，看起来像乱码）。但必须注意，只有同时保留幅度谱和相位谱，才能完整复原出原本的时域信号。

1.4 二维 DFT 公式与解析

上述公式与案例是一维信号（音频、时间序列）的 DFT 表示，而图像是二维空间信号（二维像素排列表示空间强度变化），二维的 DFT 公式类似于：

F(u,v)=∑x=0M−1∑y=0N−1f(x,y)e−j2π(uxM+vyN)F(u,v)=\sum^{M-1}_{x=0}\sum^{N-1}_{y=0}f(x,y)e^{-j2\pi(\frac{ux}{M}+\frac{vy}{N})} F(u,v)=x=0∑M−1​y=0∑N−1​f(x,y)e−j2π(Mux​+Nvy​)

  • f(x,y)f(x,y)f(x,y)：表示 空间域 的像素值（即原图）
  • F(u,v)F(u,v)F(u,v)：表示 频域 的复数结果
  • u,vu,vu,v：表示 空间频率。简单来说，u 对应图像水平方向的纹理变化快慢，v 对应垂直方向的纹理变化快慢

直接按照上述公式进行计算，其复杂度为 O(N2)O(N^{2})O(N2)（对于二维图像则是 O(M2N2)O(M^{2}N^{2})O(M2N2)）。这意味着随着图片尺寸的增加，运算量会呈平方级急剧增长。这也是为什么在实际工程中，我们几乎总是使用 快速傅里叶变换（FFT） 算法来加速这一过程的原因。

代码中，为了演示原理，我给出的两个手写函数都是严格按照公式实现的（虽然速度较慢），我也给了完整的标注。另外，对一个离散信号 x[n]x[n]x[n]，DFT 变换后得到的结果 X[k]X[k]X[k] 一般是复数：X[k]=a+jbX[k]=a+jbX[k]=a+jb，这里的 aaa 是实部，bbb 是虚部，j=−1j=\sqrt{-1}j=−1​。

幅度谱 描述的是信号在各个频率分量上的 强度，数学上是复数的模：∣X[k]∣=a2+b2|X[k]|=\sqrt{a^{2}+b^{2}}∣X[k]∣=a2+b2​，在 Numpy 中的表示就是 np.abs(X)。

相位谱 描述的是信号在各个频率上的 相位偏移，数学上是复数的角度：θ[k]=arctan⁡ba\theta{[k]}=\arctan{\frac{b}{a}}θ[k]=arctanab​，在 Numpy 中的表示就是 np.angle(X)。

import numpy as np
import matplotlib.pyplot as plt

# ---------------- 一维手写 DFT ----------------
def my_dft(x):
    N = len(x) # 获取信号长度
    # 初始化长度为 N 的复数数组，用来存储频域结果 X[k]
    X = np.zeros(N, dtype=complex) # 因为 DFT 输出是复数，所以指定 dtype=complex

    # 双重循环计算 DFT。共有 N 个频率分量，每个 k 对应一个复数 X[k]
    # 每一个复数 X[k] 对应公式中的求和部分，所有的 x[n] 的和
    for k in range(N):
        for n in range(N):
            X[k] += x[n] * np.exp(-2j * np.pi * k * n / N)
    return X

# ---------------- 创建一维信号 ----------------
N = 64
fs = 64

np.random.seed(0)
x_noise = 0.5*(np.random.rand(N)-0.5)
x_combined = x_signal + x_noise

# DFT
X_combined = my_dft(x_combined)
freq_axis = np.arange(N) * fs / N  # Hz

# ---------------- 二维手写 DFT ----------------
def my_dft2(img):
    M, N = img.shape # 获取图像的行数和列数
    F = np.zeros((M, N), dtype=complex) # 初始化二维复数矩阵存储频域结果

    # 外层两个循环对应公式中的 频率索引 u（行方向）、v（列方向）
    # 每个 (u,v) 位置最终得到一个复数 F[u,v]
    for u in range(M):
        for v in range(N):
            # 内层两个循环对应公式中的 空间索引 x（行）、y（列）
            # 对应二维 DFT 的每一项，累加求和得到 (u,v) 频率的复数
            for x in range(M):
                for y in range(N):
                    F[u,v] += img[x,y] * np.exp(-2j * np.pi * (u*x/M + v*y/N))
    return F

1.5 快速傅里叶变换（FFT）

Numpy 内置了 快速傅里叶变换（FFT） 的接口：np.fft.fft。时间复杂度为 O(Nlog⁡N)O(N\log{N})O(NlogN)，速度要快很多。FFT 优化的核心思想是 分治 + 重用，作为 Cooley–Tukey 算法的代表，它做了两个关键优化：

  1. 分解：把长度为 N 的序列拆成 偶数索引 和 奇数索引 两部分，DFT 公式做出变形：

xeven[m]=x[2m],xodd[m]=x[2m+1],m=0...(N2−1)x_{even}[m]=x[2m],x_{odd}[m]=x[2m+1], \quad m=0...(\frac{N}{2}-1) xeven​[m]=x[2m],xodd​[m]=x[2m+1],m=0...(2N​−1)

  1. 重用（对称性）：利用复指数项（旋转因子）的周期性和对称性，原本需要计算 N 个点的 DFT，被拆解成了两个 N/2 点的 DFT。

这意味着，计算 X[k] 时，只需要计算一半的项，另一半可以通过简单的加减法由前一半推导出来（即 X[k+N/2] 与 X[k] 仅差一个符号）。这种递归拆解一直进行下去，直到分解为 2 点 DFT，从而极大地减少了重复计算。

所以，速度提升了多少？如果把 DFT 和 FFT 的运算量做一个对比，你会发现随着数据量 N 的增大，FFT 的优势是碾压级的：

644,09638410倍
1,0241,048,57610,240100倍
1,000,000101210^{12}10122×1072 \times 10^72×10750,000倍

正因为 FFT 如此高效，现代所有的图像处理库（如 OpenCV、Numpy、Matlab）在处理傅里叶变换时，底层调用的全都是 FFT 算法。我们在代码中使用的 np.fft.fft2，本质上就是执行了上述的优化过程。

图 1.4 展示了二维 FFT 的实际应用。注意右图中心最亮的部分，那就是图像的直流分量（整体亮度），而向四周发散的光芒则代表了字母边缘的高频信息。

1.6 傅里叶逆变换（IFFT）

既然我们可以通过 DFT 将图像“拆解”成频率成分，那么理所当然地，我们也可以将这些频率成分“组装”回去。这就是 逆离散傅里叶变换 的作用。IFFT 是图像处理闭环的最后一步：

  1. 分析：FFT 把图像变成频谱，让我们看到噪声在哪里（比如那些周期性条纹）。
  2. 处理：我们在频域里把代表噪声的“亮点”抹掉（滤波）。
  3. 重建：IFFT 把修改后的频谱变回图像。

没有 IFFT，我们就只能看频谱图，而无法得到处理后的照片。从数学定义上来看，一维 IFFT 的公式与 DFT 非常相似，主要区别在于指数部分的符号（从 −j 变为 +j ）以及前面的归一化系数 1N\frac{1}{N}N1​：

x[n]=1N∑k=0N−1X[k]⋅ej2πNkn,n=0,1,...,N−1x[n]=\frac{1}{N}\sum^{N-1}_{k=0}X[k]\cdot{e^{j\frac{2\pi}{N}kn}}, \quad n=0,1,...,N-1 x[n]=N1​k=0∑N−1​X[k]⋅ejN2π​kn,n=0,1,...,N−1

  • X[k]X[k]X[k]：频域的复数信号（包含幅度和相位）
  • x[n]x[n]x[n]：还原后的时域信号
  • ej2πNkne^{j\frac{2\pi}{N}kn}ejN2π​kn：注意这里变成了正号，表示反向旋转

对于二维图像，公式同理扩展为双重求和：

f(x,y)=1MN∑u=0M−1∑v=0N−1F(u,v)ej2π(uxM+vyN)f(x,y)=\frac{1}{MN}\sum^{M-1}_{u=0}\sum^{N-1}_{v=0}F(u,v)e^{j2\pi(\frac{ux}{M}+\frac{vy}{N})} f(x,y)=MN1​u=0∑M−1​v=0∑N−1​F(u,v)ej2π(Mux​+Nvy​)

在 Numpy 中，我们使用 np.fft.ifft2 来处理图像。这里有一个非常关键的细节：虽然原图是纯实数（像素值），但 IFFT 计算出来的结果通常是复数。你可能会问：“为什么还原出来的图像会有虚部？”

  • 理论原因：如果频域数据完美对称（共轭对称），逆变换的结果虚部应该全为 0。
  • 实际原因：由于计算机的浮点数精度误差，计算过程中会产生极其微小的虚部数值（例如 10−1610^{−16}10−16 级别）。

因此，在显示或保存图像时，我们通常只取结果的实部：

# 1. 逆傅里叶变换（假设 F_shift 是处理过的频域数据）
img_back = np.fft.ifft2(F_shift)

# 2. 将频谱移回角落（与 fftshift 对应）
img_back = np.fft.ifftshift(img_back)

# 3. 取实部（去除微小的计算误差虚部），此时 img_back 就是还原后的空间域图像
img_back = np.real(img_back)

2. 噪声与频域滤波

在深入具体的滤波器之前，我们需要先建立一个宏观的认知：什么是频率滤波？简单来说，频率滤波就是通过数学手段，在频域中对信号的频率成分进行“筛选”或“修改”的过程。想象你面前有一盘混合了各种颜色豆子（不同频率成分）的盘子。

  • 时域：你只能看到豆子混在一起的样子。
  • 频域：你把豆子按颜色（频率）分拣开了。
  • 滤波：你根据规则，把不想要的颜色扔掉，把想要的颜色留下来。

2.1 频域滤波步骤

在 OpenCV 或数字图像处理中，进行频率滤波通常遵循以下标准步骤（四步走）：

  1. 傅里叶变换：将图像从空间域转换到频率域。

    • 公式：F(u,v)=F{f(x,y)}F(u,v)=\mathcal{F}\{f(x,y)\}F(u,v)=F{f(x,y)}
    • 操作：使用 cv2.dft()。
    • 结果：得到包含幅度和相位信息的复数频谱。

  2. 频谱中心化：将频谱的 零频率分量（直流分量） 移到频谱图的中心。

    • 目的：为了方便设计滤波器（通常滤波器是关于中心对称的）。
    • 操作：使用 np.fft.fftshift。

  3. 频域相乘（滤波核心）：将中心化后的频谱 F(u,v)F(u,v)F(u,v) 与 滤波器函数 H(u,v)H(u,v)H(u,v) 进行逐点相乘（为什么频域处理比卷积快？因为它的滤波器和图像一样大，直接相乘）。

    • 公式：G(u,v)=F(u,v)⋅F(u,v)G(u,v)=F(u,v)\cdot{F(u,v)}G(u,v)=F(u,v)⋅F(u,v)
    • 原理：
      • 如果 H(u,v)=1H(u,v)=1H(u,v)=1，该频率成分保留。
      • 如果 H(u,v)=0H(u,v)=0H(u,v)=0，该频率成分被“杀掉”。
      • 如果 0<H(u,v)<10<H(u,v)<10<H(u,v)<1，该频率成分被“减弱”。

  4. 逆傅里叶变换：将处理后的频谱 G(u,v)G(u,v)G(u,v) 转换回空间域，得到处理后的图像 g(x,y)g(x,y)g(x,y)。

    • 公式：g(x,y)=F−1{G(u,v)}g(x,y)=\mathcal{F}^{-1}\{G(u,v)\}g(x,y)=F−1{G(u,v)}
    • 操作：使用 cv2.idft()。

2.2 OpenCV函数解析

cv2.dft(src, dst=None, flags=0, nonzeroRows=0)

  • src：输入图像。必须是 np.float32 类型（不能是 uint8）。

  • flags：这是一个位掩码（Bitmask），用于控制变换的行为。你可以组合使用多个标志（例如 cv2.DFT_INVERSE | cv2.DFT_SCALE）。

    • cv2.DFT_COMPLEX_OUTPUT：输出复数。最常用。默认情况下 OpenCV 可能会优化输出，但为了获取完整的频谱（实部+虚部），必须强制指定此标志。
    • cv2.DFT_INVERSE：逆变换。用于从频域还原回空间域（即 idft 的功能）。
    • cv2.DFT_SCALE：缩放。通常在逆变换时使用，将结果除以 N，使像素值回到 0-255 范围。
    • cv2.DFT_ROWS：按行变换。对每一行进行一维 DFT，用于处理多维数据或特定优化。
    • cv2.DFT_REAL_OUTPUT：输出实数。强制输出实数（通常用于逆变换后）。

  • nonzeroRows：优化参数。如果你知道输入矩阵只有前几行有数据，可以指定行数来加速计算。一般情况下设为 0（默认）。

cv2.idft(src, dst=None, flags=0)
  • src (必填)：输入的复数矩阵（通常是 DFT 的结果）。必须是 np.complex64 或者双通道浮点数（实部+虚部）。
  • flags（选填）：控制标志。虽然 idft 默认就是做逆变换，但这里可以加“佐料”。
    • DFT_SCALE：缩放。因为逆变换后数值会变大 N 倍，加这个标志会自动除以 N，让像素值回到 0-255 范围。
    • DFT_REAL_OUTPUT：只输出实部。因为理论上逆变换结果应该是实数（虚部接近 0），加这个标志可以直接扔掉虚部，省去后面写 magnitude 或取 [:,:,0] 的麻烦。
magnitude = cv2.magnitude(x, y[, magnitude])
  • x (InputArray)：输入矩阵的“实部”。单通道的浮点型矩阵（DFT 结果中的第 0 通道）。在几何意义上，它代表复数向量的水平分量。
  • y (InputArray)：输入矩阵的“虚部”。单通道的浮点型矩阵（DFT 结果中的第 1 通道）。它必须和参数 x 具有完全相同的尺寸和类型。在几何意义上，它代表复数向量的垂直分量。
  • magnitude（输出）：计算出的幅值矩阵。每个像素的值代表了该频率分量的“强度”或“能量”。本质上是利用勾股定理计算每个对应元素，magnitude(I)=x(I)2+y(I)2magnitude(I)=\sqrt{x(I)^{2}+y(I)^{2}}magnitude(I)=x(I)2+y(I)2​。

2.3 低通滤波演进

  1. 理想低通滤波器 (ILPF)

H(u,v)={1,if D(u,v)≤D00,if D(u,v)>D0H(u,v) = \begin{cases} 1, & \text{if } D(u,v) \leq D_0 \\ 0, & \text{if } D(u,v) > D_0 \end{cases} H(u,v)={1,0,​if D(u,v)≤D0​if D(u,v)>D0​​

这是最“霸道”的滤波器，像一把刀一样直接切断高频。D(u,v)D(u,v)D(u,v) 是频率点到中心的距离，D0D_0D0​ 是截止频率（半径）。在半径内完全保留，半径外完全剔除。因为过渡太生硬，会导致图像出现 振铃效应（像水波纹一样的伪影）。

  1. 巴特沃斯低通滤波器 (BLPF)

H(u,v)=11+[D(u,v)/D0]2nH(u,v) = \frac{1}{1 + [D(u,v) / D_0]^{2n}} H(u,v)=1+[D(u,v)/D0​]2n1​

这是最“圆滑”的折中方案，也是工程中最常用的。n 是滤波器的阶数。它没有明显的截止界限，而是随着距离增加平滑衰减。当 n 越大，它越接近理想低通；当 n 较小时，过渡非常平缓。它通常能避免振铃效应。

  1. 高斯低通滤波器 (GLPF)

H(u,v)=e−D(u,v)2/2D02H(u,v) = e^{-D(u,v)^2 / 2D_0^2} H(u,v)=e−D(u,v)2/2D02​

这是最“自然”的滤波器，也是 OpenCV 默认喜欢的类型。它的形状就是标准的高斯分布（钟形曲线）。它的反变换还是高斯函数，这意味着它在空间域和频域都是平滑的。它完全不会产生振铃效应，模糊效果最柔和。

import cv2
import numpy as np
import matplotlib.pyplot as plt

# ---------------- 1. 图像预处理与构建滤波器（DFT） ----------------

# 以灰度模式读取图像，获取图像尺寸，计算中心点坐标
img = cv2.imread('img_01.png', 0)
rows, cols = img.shape
crow, ccol = rows // 2, cols // 2

# 图像转换为浮点型，执行 DFT，并将零频率分量移到频谱中心
img_float = np.float32(img)
dft = cv2.dft(img_float, flags=cv2.DFT_COMPLEX_OUTPUT)
dft_shift = np.fft.fftshift(dft)

# 生成距离矩阵 D(u,v)，设置截止频率 D0
u = np.arange(rows) - crow  # 行坐标 - 中心行坐标，[−256,−255,...,0,...,255]^T
v = np.arange(cols) - ccol  # 列坐标 - 中心列坐标，[−256,−255,...,0,...,255]
U, V = np.meshgrid(v, u)    # 通过广播生成二维坐标矩阵 U 和 V，分别表示每个频率点的水平和垂直距离
D = np.sqrt(U**2 + V**2)    # 计算每个频率点到中心的距离 D(u,v)，得到一个与图像尺寸相同的矩阵 D
D0 = 50  # 截止频率

# 1. 理想低通滤波器 (Ideal Low-Pass Filter)
mask_ideal = np.zeros((rows, cols), np.float32)
mask_ideal[D <= D0] = 1

# 2. 巴特沃斯低通滤波器 (Butterworth Low-Pass Filter)
n = 2 # 阶数
mask_butter = 1 / (1 + (D / D0)**(2*n))

# 3. 高斯低通滤波器 (Gaussian Low-Pass Filter)
mask_gaussian = np.exp(-(D**2) / (2 * D0**2))

# ---------------- 2. 通用处理与绘图函数 ----------------
def apply_filter_process(mask, dft_shift):
    """应用滤波器并返回：滤波后频谱(对数), 还原后的图像"""
    if mask is None:
        filtered_shift = dft_shift  # 原始信号处理
    else:
        mask_2ch = np.stack([mask, mask], axis=-1)  # 扩展通道以匹配复数矩阵
        filtered_shift = dft_shift * mask_2ch  # 频域相乘

    # 计算幅度谱 (用于显示)
    magnitude = cv2.magnitude(filtered_shift[:,:,0], filtered_shift[:,:,1])
    mag_log = 20 * np.log(magnitude + 1e-6)

    # 逆变换 (用于显示图像)
    f_ishift = np.fft.ifftshift(filtered_shift)
    img_back = cv2.idft(f_ishift)
    img_back = cv2.magnitude(img_back[:,:,0], img_back[:,:,1])
    img_back = cv2.normalize(img_back, None, 0, 255, cv2.NORM_MINMAX)

    return mag_log, img_back.astype(np.uint8)

# 存放所有滤波器的字典，方便循环处理
filters = {
    'Original': None, # 原始信号不需要 Mask
    'Ideal': mask_ideal,
    'Butterworth': mask_butter,
    'Gaussian': mask_gaussian
}

# 设置 Matplotlib 字体和显示参数，确保中文显示正常
plt.rcParams['font.sans-serif']=['Arial Unicode MS']
plt.rcParams['axes.unicode_minus']=False

def show_img(data, title, pos, cmap='gray'):
    """通用绘图函数"""
    plt.subplot(3, 4, pos)
    plt.imshow(data, cmap=cmap)
    plt.title(title, fontsize=12) # 稍微调大字体
    plt.axis('off') # 隐藏坐标轴
    plt.grid(False)

# ---------------- 4. 绘制 3行 4列 对比图 ----------------
fig = plt.figure(figsize=(18, 12))
col_names = ['Original', 'Ideal', 'Butterworth', 'Gaussian']

# --- 第1行：原图和滤波器掩膜 (Filter Mask) ---
show_img(img, "1. 原始图像 (Original Image)", 1)
show_img(mask_ideal, "4. 理想滤波器掩膜 (Ideal Mask)", 2)
show_img(mask_butter, "7. 巴特沃斯滤波器掩膜 (Butterworth Mask)", 3)
show_img(mask_gaussian, "10. 高斯滤波器掩膜 (Gaussian Mask)", 4)

# --- 第2行：频域幅度谱 (Frequency Spectrum) ---
# 循环处理每一列
for i, name in enumerate(col_names):

    mag_log, _ = apply_filter_process(mask, dft_shift)  # 计算频谱
    pos = 4 + (i + 1)  # 第 2 行是 4 + (i+1)

        'Original': '2. 原始幅度谱 (Original Spectrum)',
        'Ideal': '5. 理想滤波频谱 (Ideal Spectrum)',
        'Butterworth': '8. 巴特沃斯滤波频谱 (Butterworth Spectrum)',
        'Gaussian': '11. 高斯滤波频谱 (Gaussian Spectrum)'
    }

        'Butterworth': '9. 巴特沃斯还原图 (折中效果)',

2.4 高通滤波演进

高通滤波可以由低通滤波简单变换得到。简单来说

  1. 演进逻辑：完全一样（理想 -> 巴特沃斯 -> 高斯）。
  2. 振铃效应：完全一样（硬切就有振铃，平滑就没有）。
  3. 数学公式：不是“一样”，而是“相反”。高通（HPF） = 1 - 低通（LPF）。

逻辑很简单：全图是 1，减去保留的部分（低通），剩下的就是去掉的部分（高通）。假设 D(u,v)D(u,v)D(u,v) 是距离，D0D_{0}D0​ 是截止频率，在数学公式上

  • 理想高通：直接反转，0 变 1，1 变 0。

H(u,v)={0,if D(u,v)≤D01,if D(u,v)>D0H(u,v) = \begin{cases} 0, & \text{if } D(u,v) \leq D_0 \\ 1, & \text{if } D(u,v) > D_0 \end{cases} H(u,v)={0,1,​if D(u,v)≤D0​if D(u,v)>D0​​

  • 巴特沃斯高通：分子分母颠倒 (本质也是 1 - LP)

H(u,v)=11+[D0/D(u,v)]2nH(u,v) = \frac{1}{1 + [D_0 / D(u,v)]^{2n}} H(u,v)=1+[D0​/D(u,v)]2n1​

  • 高斯高通：1 减去原式

H(u,v)=1−e−D(u,v)2/2D02H(u,v) = 1 - e^{-D(u,v)^2 / 2D_0^2} H(u,v)=1−e−D(u,v)2/2D02​

高通滤波器其实就是低通滤波器的底片（反相）。低通是用来磨皮（去噪）的，高通就是用来整容（锐化/提取边缘）的。同样的，越‘理想’的刀法，留下的疤痕（振铃）越重；越‘高斯’的刀法，处理越柔和。

import cv2
import numpy as np
import matplotlib.pyplot as plt

# ---------------- 1. 图像预处理与构建滤波器（DFT） ----------------
img = cv2.imread('img_01.png', 0)
rows, cols = img.shape
img_float = np.float32(img)
# 执行 DFT，这里没有 fftshift，低频分量自然位于四个角
dft = cv2.dft(img_float, flags=cv2.DFT_COMPLEX_OUTPUT)

u = np.arange(rows)
v = np.arange(cols)
U, V = np.meshgrid(v, u)

# 计算到四个角的距离：左上(0,0), 右上(0, cols), 左下(rows, 0), 右下(rows, cols)
# 我们取这四个距离中的最小值，作为该点到“低频区”的距离
dist_tl = np.sqrt((U - 0)**2 + (V - 0)**2)
dist_tr = np.sqrt((U - (cols-1))**2 + (V - 0)**2)
dist_bl = np.sqrt((U - 0)**2 + (V - (rows-1))**2)
dist_br = np.sqrt((U - (cols-1))**2 + (V - (rows-1))**2)

# 核心逻辑：取最小值。比如一个点在左上角，dist_tl 最小，说明它属于低频，要被过滤掉
D = np.minimum(np.minimum(dist_tl, dist_tr), np.minimum(dist_bl, dist_br))
D0 = 30  # 截止频率（挖掉四角的半径）

# 1. 理想高通滤波器 (Ideal High-Pass Filter)
mask_ideal = np.zeros((rows, cols), np.float32)
mask_ideal[D > D0] = 1

# 2. 巴特沃斯高通滤波器 (Butterworth High-Pass Filter)
n = 2
mask_butter = 1 / (1 + (D0 / (D + 1e-6))**(2*n)) # 1e-6 避免除零

# 3. 高斯高通滤波器 (Gaussian High-Pass Filter)
mask_gaussian = 1 - np.exp(-(D**2) / (2 * D0**2))

# ---------------- 2. 通用处理与绘图函数 ----------------
def apply_filter_process(mask, dft):
    """应用滤波器并返回：滤波后频谱(对数), 还原后的图像"""
    if mask is None:
        filtered_dft = dft
    else:
        # 扩展通道：mask是(512,512)，dft是(512,512,2)
        mask_2ch = np.stack([mask, mask], axis=-1)
        filtered_dft = dft * mask_2ch

    # 逆变换，因为没做 fftshift，所以这里也不需要做 ifftshift
    img_back = cv2.idft(filtered_dft)
    img_back = cv2.magnitude(img_back[:,:,0], img_back[:,:,1])
    img_back = cv2.normalize(img_back, None, 0, 255, cv2.NORM_MINMAX)

    return img_back.astype(np.uint8)

# 存放所有滤波器的字典
filters = {
    'Original': None,
    'Ideal HPF': mask_ideal,
    'Butterworth HPF': mask_butter,
    'Gaussian HPF': mask_gaussian
}

# 设置绘图参数
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

def show_img(data, title, pos, cmap='gray'):
    """通用绘图函数"""
    plt.subplot(2, 4, pos)
    plt.imshow(data, cmap=cmap)
    plt.title(title, fontsize=12)
    plt.axis('off')
    plt.grid(False)

# ---------------- 3. 绘制 3行 4列 对比图 ----------------
fig = plt.figure(figsize=(14, 8))
col_names = ['Original', 'Ideal HPF', 'Butterworth HPF', 'Gaussian HPF']

# --- 第1行：原图和滤波器掩膜 (Filter Mask) ---
show_img(img, "1. 原始图像", 1)
show_img(mask_ideal, "3. 理想高通掩膜 (中间白，四角黑)", 2)
show_img(mask_butter, "5. 巴特沃斯高通掩膜", 3)
show_img(mask_gaussian, "7. 高斯高通掩膜", 4)

# --- 第3行：空间域还原图像 (Restored Image) ---
for i, name in enumerate(col_names):
    mask = filters[name]
    img_back = apply_filter_process(mask, dft)
    pos = 4 + (i + 1)

    title_map = {

        'Ideal HPF': '4. 理想高通结果 (注意：边缘有振铃)',
        'Butterworth HPF': '6. 巴特沃斯结果',
        'Gaussian HPF': '8. 高斯高通结果 (边缘柔和)'
    }
    show_img(img_back, title_map[name], pos)

plt.tight_layout()
plt.show()

2.5 带阻/带通滤波

带阻（Band-Stop） 和 带通（Band-Pass） 就是低通和高通的“进阶版”。在数学关系上：带通 = 1 - 带阻（和低通/高通的关系一模一样）。可以把它们看作是“切蛋糕”的不同切法：

滤波器类型形象比喻作用频域里的样子 (中心化后)
低通 (LPF)只留圆心让低频通过，挡住高频中间白，四周黑
高通 (HPF)只挖圆心挡住低频，让高频通过中间黑，四周白
带阻 (BSF)挖个圈挡住特定频段，其他都过中间白，画个黑圈，外围白
带通 (BPF)只留圈只让特定频段过，其他都挡中间黑，画个白圈，外围黑
  1. 理想带阻滤波器 (Ideal Bandreject Filter)：最简单粗暴，以 D0D_0D0​ 为中心，宽度为 WWW 的黑圈（阻带），其余部分全为 1（通带）。

import numpy as np
import matplotlib.pyplot as plt

def get_band_filter_mask(shape, d0, w, filter_type='ideal', filter_mode='reject'):
    """
    生成带阻或带通滤波器 Mask
    :param shape: 图像尺寸 (rows, cols)
    :param d0: 中心频率半径
    :param w: 频带宽度
    :param filter_type: 'ideal', 'butterworth', 'gaussian'
    :param filter_mode: 'reject' (带阻) or 'pass' (带通)
    """
    rows, cols = shape
    crow, ccol = rows // 2, cols // 2

    # 生成距离矩阵 D(u,v)
    u = np.arange(rows) - crow
    v = np.arange(cols) - ccol
    V, U = np.meshgrid(v, u) # 注意 meshgrid 的 xy 顺序
    D = np.sqrt(U**2 + V**2) + 1e-8 # 加极小值防止除零

    # 1. 计算带阻滤波器 (Base)
    if filter_type == 'ideal':
        mask_bs = np.ones((rows, cols))
        mask_bs[(D >= (d0 - w/2)) & (D <= (d0 + w/2))] = 0
    elif filter_type == 'butterworth':
        n = 2
        numerator = D * w
        denominator = D**2 - d0**2
        denominator[denominator == 0] = 1e-8  # 处理分母为0的情况
        mask_bs = 1 / (1 + (numerator / denominator)**(2*n))
    elif filter_type == 'gaussian':
        exponent = -0.5 * ((D**2 - d0**2) / (D * w))**2
        mask_bs = 1 - np.exp(exponent)
    else:
        raise ValueError("Unsupported filter type")

    # 2. 根据模式返回 'reject' (带阻) or 'pass' (带通)
    if filter_mode == 'reject':
        return mask_bs
    else:
        return 1 - mask_bs

# --- 主程序 ---

2.6 周期性噪声的去噪

在《噪声与平滑化》章节说过，噪声的两大来源，一种是传感器（CCD、CMOS）产生的，另一种是传输介质产生的。周期性噪声（Periodic Noise）也不列外：

  • 读出噪声 (Read Noise)：这是传感器在读取每个像素的电荷信号时，由内部的放大器和模数转换器（ADC）等电路产生的电子噪声。这种噪声与传感器的读出时序紧密相关，因此可能表现出周期性的特征。

  • 50Hz 工频干扰（传输介质）：我们周围的电网（50Hz或60Hz）会产生强大的电磁场。如果信号传输线路的屏蔽做得不好，或者电路板的接地设计不合理，就很容易“拾取”到这种干扰。

    • 这种干扰会像一个正弦波一样，被“调制”到图像信号上。
    • 最终在图像上就表现为明暗相间的、有规律的条纹或滚动条。

为什么说带阻滤波对去除周期性噪声很好用？因为：周期性噪声 = 特定的频率。

  1. 现象：假设你的图片上有一堆横着的条纹（比如扫描线），这在数学上就是一个正弦波。

  2. 频域表现：当你做傅里叶变换后，这个正弦波不会像图像主体那样聚在中心，而是会变成频谱图上对称的几个极亮的点。这就好比在茫茫大海（图像频谱）里，突然竖起了几根高耸入云的柱子（噪声频谱）。

  3. 带阻滤波器的作用：我们不需要像低通滤波那样把高频全切了（那样图像会糊），只需要做一个“陷波滤波器”（Notch Filter，一种特殊的带阻滤波器），把那几根“柱子”打掉（像素值设为0）即可。

import numpy as np
import matplotlib.pyplot as plt
import cv2

# ==========================================
# 1. 核心算法模块：高斯陷波滤波器生成器
# ==========================================
class GaussianNotchFilter:
    @staticmethod
    def create(shape, noise_coords, sigma=5):
        """
        生成高斯陷波滤波器掩模
        :param shape: 图像尺寸 (rows, cols)
        :param noise_coords_list: 噪声坐标列表 [(y1, x1), (y2, x2), ...]
        :param sigma: 陷波半径/平滑度
        :return: 滤波器掩模 H (0~1)
        """
        rows, cols = shape
        crow, ccol = rows // 2, cols // 2
        ny, nx = noise_coords

        # 创建网格坐标
        u = np.arange(rows) - crow
        v = np.arange(cols) - ccol
        V, U = np.meshgrid(v, u)

        # 计算到噪声点及其对称点的欧氏距离
        # 噪声点 1，噪声点 2 (共轭对称点)
        u1, v1 = ny - crow, nx - ccol
        D1 = np.sqrt((U - u1)**2 + (V - v1)**2)
        u2, v2 = -u1, -v1
        D2 = np.sqrt((U - u2)**2 + (V - v2)**2)

        # 高斯陷波公式：H = 1 - (Gaussian1 + Gaussian2)，高斯比硬截断更平滑
        notch1 = np.exp(-(D1**2) / (2 * sigma**2))
        notch2 = np.exp(-(D2**2) / (2 * sigma**2))

        H = 1 - (notch1 + notch2)
        return np.clip(H, 0, 1)

# ==========================================
# 2. 可视化模块
# ==========================================
def plot_results(images_dict):
    """
    通用绘图函数
    :param images_dict: 字典 {标题: 图像数组, ...}
    """
    plt.rcParams['font.sans-serif'] = ['Arial Unicode MS'] # 兼容中文显示
    plt.figure(figsize=(15, 8))

    for i, (title, img) in enumerate(images_dict.items()):
        plt.subplot(2, 3, i + 1)
        # 自动判断颜色映射：如果是频谱用 jet，其他用 gray
        cmap = 'jet' if '频谱' in title else 'gray'
        plt.imshow(img, cmap=cmap)
        plt.title(title)
        plt.axis('off')

    plt.tight_layout()
    plt.show()

# ==========================================
# 3. 主程序流程
# ==========================================
def main():
    # --- 1. 读取与预处理 ---
    img = cv2.imread('pNoise.png', 0)
    if img is None:
        print("错误：未找到 pNoise.png")
        return

    rows, cols = img.shape
    crow, ccol = rows // 2, cols // 2

    # --- 2. 傅里叶变换 ---
    dft = np.fft.fft2(img)
    dft_shift = np.fft.fftshift(dft)
    magnitude_spectrum = 20 * np.log(np.abs(dft_shift) + 1e-5)

    # --- 3. 自动定位噪声频率 ---
    # 构建中心掩模，屏蔽低频主体
    mask_center = np.zeros((rows, cols), np.uint8)
    cv2.circle(mask_center, (ccol, crow), int(rows/10), 255, -1)

    # 寻找高频区的最大能量点
    temp_spectrum = magnitude_spectrum.copy()
    temp_spectrum[mask_center == 255] = 0
    ny, nx = np.unravel_index(np.argmax(temp_spectrum), temp_spectrum.shape)

    # 计算半径 r
    r = int(np.sqrt((nx - ccol)**2 + (ny - crow)**2))
    print(f"🎯 检测到噪声频率: ({nx}, {ny}), 半径 r={r}")

    # --- 4. 生成滤波器 (调用封装好的类) ---
    # 这里可以随意调整 sigma 参数
    H_notch_30 = GaussianNotchFilter.create((rows, cols), (ny, nx), sigma=30)
    H_notch_60 = GaussianNotchFilter.create((rows, cols), (ny, nx), sigma=60)
    H_notch_90 = GaussianNotchFilter.create((rows, cols), (ny, nx), sigma=90)

    # --- 5. 滤波与还原 ---
    filtered_dft_30 = dft_shift * H_notch_30
    filtered_dft_60 = dft_shift * H_notch_60
    filtered_dft_90 = dft_shift * H_notch_90
    img_back_30 = np.abs(np.fft.ifft2(np.fft.ifftshift(filtered_dft_30)))
    img_back_60 = np.abs(np.fft.ifft2(np.fft.ifftshift(filtered_dft_60)))
    img_back_90 = np.abs(np.fft.ifft2(np.fft.ifftshift(filtered_dft_90)))

    # --- 6. 绘图 (调用封装好的函数) ---
    results = {
        '1. 原始含噪图': img,
        f'2. 陷波滤波器 (r={r}; sigma=30)': H_notch_30,
        '3. 滤波后频谱': 20 * np.log(np.abs(filtered_dft_30) + 1e-5),
        '4. 去噪结果 (sigma=30)': img_back_30,
        '5. 去噪结果 (sigma=60)': img_back_60,
        '6. 去噪结果 (sigma=90)': img_back_90
    }
    plot_results(results)

if __name__ == '__main__':
    main()

2.7. 扩展：为什么直流分量又叫做零频率分量？

简单来说，“直流分量”之所以叫“零频率分量”，是因为它代表了信号中“完全不动”的那一部分。我们从数学定义和物理直观两个角度把它彻底讲透：

  1. 数学角度：公式里的“0”

傅里叶变换的核心是把信号分解成一个个正弦波。正弦波的通用公式是：

y=A⋅cos⁡(2πft)y = A\cdot{\cos(2\pi ft)} y=A⋅cos(2πft)

其中 f 是频率。现在，我们来看看当 频率 f=0 时会发生什么：

y=A⋅cos⁡(2π⋅0⋅t)=A⋅cos⁡(0)=A⋅1=Ay = A\cdot{\cos(2\pi\cdot{0}\cdot{t})} = A\cdot{\cos(0)} = A\cdot{1} = A y=A⋅cos(2π⋅0⋅t)=A⋅cos(0)=A⋅1=A

当频率为 0 时，正弦波不再是波浪线，而变成了一条恒定的直线（常数 A ）。在信号处理中，这个不随时间变化的常数，就是直流分量。所以，零频率分量 = 直流分量。

  1. 物理直观：平均值
  • 交流 (AC)：指的是像波浪一样忽高忽低、不断变化的部分。
  • 直流 (DC)：指的是像电池一样，电压稳定不变的部分。

如果你把一张图片的所有像素值加起来除以总像素数，得到一个“平均灰度值”。这个值代表了图片的整体亮度基调。

  • 无论图片里的花纹（高频）怎么变，这个平均值（直流）是不动的，它不震荡。
  • 在频谱图上，这个平均值就体现在坐标原点（0Hz）的那一个点上。
