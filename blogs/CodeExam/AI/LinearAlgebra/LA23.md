#### 难度：中等

实现一个函数，将给定矩阵转换为简化行阶梯形（Reduced Row Echelon Form, RREF）。RREF 是一种特殊的矩阵形式，具有以下特点：

1. 每行的首个非零元素（主元）为 1
2. 主元所在列的其他元素都为 0
3. 每行主元右侧的零行都在非零行之下

- **输入描述**：函数`rref`接收一个参数：
  1. matrix：numpy 数组表示的矩阵
- **输出描述**：返回对应的简化行阶梯形，类型为 numpy 数组。

#### 解题思路：Python

```py
def matrixmul(A, B):
    """手写矩阵乘法 A x B"""
    m, n, p, q = len(A), len(A[0]), len(B[0]), len(B)
    if m != p or n != q:
        return -1

    C = [[0] * p for _ in range(m)]
    for i in range(m):
        for j in range(p):
            for k in range(n):
                C[i][j] += A[i][k] * B[k][j]
    return C
```
